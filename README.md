# EdTech Courses Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes EdTech course data and provides insights into ratings, enrollments, categories, languages, and instructor performance.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Key Insights
- Course Rating Analysis
- Enrollment Trends
- Category-wise Distribution
- Language-wise Analysis
- Instructor Performance
